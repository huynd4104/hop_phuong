# Warm Claymorphism - UI Skill System 🎨✨

This system enforces a consistent, premium, 3D, and warm "Claymorphism" UI/UX design language across all generating AI agents.

Use this folder to give any AI agent a "Design Skill" so it stops producing generic, sharp, cold UI, and instead produces extremely beautiful, modern, soft interfaces.

## 🤖 Chú ý cho AI Agent
> **[AI AGENT: STOP. IF YOU ARE READING THIS, YOU MUST IMMEDIATELY READ `AI_SYSTEM_RULES.md` BEFORE PROCEEDING.]**

## 🚀 How to Use (Dành cho con người)

Bạn không cần phải copy một prompt cực dài nữa. Nhờ hệ thống dẫn đường tự động bên trong thư mục này, tất cả những gì bạn cần làm là gõ một câu lệnh ngắn ngọn để AI (Cursor, Windsurf...) hiểu và tự động đi theo nguyên tắc:

> **"Hãy thiết kế/sửa lại giao diện trang `Home.tsx` theo `ui-skill-system`"**

Khi đó: AI sẽ tự động đọc `README.md`, bị chuyển hướng tải file `AI_SYSTEM_RULES.md`, từ đó nó hiểu được nguyên tắc "Bảo vệ Logic cốt lõi", tự động tra cứu file màu sắc, đổ bóng và Mega-prompt để hoàn thành giao diện.

The AI will output React/Tailwind code that perfectly matches the warm, soft, floating, rounded, and highly polished aesthetic of this skill system.

## 📁 System Structure

- **`02-tokens/`**: The precise hex colors, radii, and `tailwind.config.js` to ensure the exact earthy Terracotta and Beige combinations.
- **`03-prompts/`**: **THE MEGA-PROMPT**. The most important file. Feed this directly to agents.
- **`04-examples/`**: Boilerplates for React components so the AI has 100% correct code to mimic.

## 🌟 Key Aesthetic Features

1. **Warm Palette**: Terracotta (`#C98C7B`), Beige Backgrounds (`#F6F1EC`), deep brown text.
2. **Abundant Roundness**: Minimum `16px` for small elements, `32px` for normal cards, `9999px` (pill) for buttons.
3. **Floating UI**: Cards do not touch screen edges. Everything hovers via soft shadows.
4. **Tactile Interaction**: Elements lift on hover and depress (inner-shadow) on click.
